# FrontendAngular

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.3.1.

## Setup
`git clone https://github.com/AlejandroWilcke/frontend-angular.git`

`cd frontend-angular`

`npm i`

## Development server

`ng serve`

## URL app
`http://localhost:4200`

## Test app
`ng test`
